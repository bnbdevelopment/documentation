title: Tasks Components
---

Ez a szekció a feladat komponensek dokumentációit tartalmazza.

## Komponensek

- `Task-Header` - Feladat fejléc és leírás.
- `Table` - Táblázatos feladat megoldás.
- `Connect` - Párosítós feladat.
- `Fill` - Kiegészítős feladat.
- `Quiz` - Kvíz kérdések és válaszok.
- `PoemShow` - Vers megjelenítése modálban.
- `PoemLine` - Verssoros feladat.
- `Flashcards` - Szókártya tanulás.
- `Flashcard` - Egyedi szókártya.
- `FlashcardCounter` - Szókártya számláló.
- `Grade` - Érdemjegy/eredmény kijelzés.
- `Suggestion` - Javasolt feladatok.
- `task-posts` - Tételek listázása.
- `AutoText` - Rövid szöveg automatikus megjelenítése.

## Típusok

- `Types` - Feladat komponensek prop és adat típusai.
