title: Irodalomerettsegi Components
---

Ez a szekció az Irodalomerettsegi komponensek dokumentációit tartalmazza, kategóriák szerint rendezve.

## Kategóriák

- `admin` - Admin felület komponensei és típusaik.
- `content` - Blog, komment és tartalom megjelenítés.
- `core` - Alap működési komponensek (cookie, markdown).
- `feedback` - Hibák, siker üzenetek, visszajelzés és reakciók.
- `file` - Fájl feltöltés és megjelenítés.
- `layout` - Oldal struktúra és modálok.
- `share` - Megosztás és eredménykártyák.
- `tasks` - Feladat típusok és interaktív elemek.
- `ui` - Általános UI komponensek.
- `user` - Felhasználói profilhoz kapcsolódó elemek.
