title: User Components
---

Ez a szekció a felhasználói komponensek dokumentációit tartalmazza.

## Komponensek

- `ProfileHeader` - Felhasználó név és email megjelenítése.
- `ProfileStats` - Felhasználói statisztikák összegzése.
- `ProfileActions` - Profilhoz tartozó műveletek.
- `Profile_Icon` - Felhasználói avatar/ikon.
- `Profile_Button` - Profil gomb, gyors műveletekkel.
- `Streak` - Napi aktivitás sorozat kijelzése.
- `Login` - Bejelentkezési link/fejléc maradvány.
