# `ProfileHeader.tsx`

Ez a komponens a felhasználó nevét, email címét és profilképét jeleníti meg.

## Props-ok

| Prop     | Típus  | Leírás                           |
|----------|--------|------------------------------------|
| `username` | `string` | A felhasználó neve.               |
| `email`    | `string` | A felhasználó email címe.           |

## Példa

```tsx
import ProfileHeader from './ProfileHeader';

<ProfileHeader username="Teszter" email="teszter@example.com" />
```
