---
title: Admin Components
---

Ez a szekció az admin komponensek dokumentációit tartalmazza.

## Komponensek

- `Admin-menu` - Admin felület menü és layout.
- `StatusAdmin` - Admin státusz kijelzés.
- `Stats` - Admin statisztikák.
- `Message` - Vélemény/üzenet elem.

## Típusok

- `Types` - Admin prop és adat típusok.
