title: UI Components
---

Ez a szekció a UI komponensek dokumentációit tartalmazza.

## Komponensek

- `Card` - Kártya megjelenítés címmel, leírással és ikonnal.
- `Select` - Választóelem (kártya-szerű).
- `Progress` - Haladás kijelző.
- `Loader` - Betöltési állapot jelzése.
- `Skeleton` - Helyőrző elem betöltéshez.
- `NextButton` - Navigációs/folytatás gomb.
- `InProgress` - Folyamatban jelzés.
- `ShowHints` - Tippek megjelenítése.
- `TitleFromId` - Dinamikus cím ID alapján.
- `Swipe` - Húzós interakciók.
- `categories-cards` - Kategória kártya rács.
- `ads` - Hirdetések komponensei.

## Típusok

- `Types` - UI komponensek prop és segéd típusai.
